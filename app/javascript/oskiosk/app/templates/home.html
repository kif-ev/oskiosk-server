Home

Hello and welcome to the Oskiosk web interface! In the future we will show you information about recent customer activity here. For now, we just include a short explanation about what to find on which page.

Administrator views

Products

Create or modify products. Manage prices, barcodes, etc.

Users

Create or modify users. Manage names, ID badge barcodes, etc.

Employee views

Sales Point

This view can be used by a kiosk employee to handle customer checkouts. Should be used with a barcode scanner.

Cash Point

This view can be used by a kiosk employee to handle customer payments. Should be used with a barcode scanner.

Customer views

Self-service Point

This view can be deployed to a kiosk terminal. Customers can checkout products on their own. Limited functionality and different UI. Should be used with a barcode scanner.
